# Specification

## Summary
**Goal:** Apply premium visual redesign to Oishi Heal Touch healthcare website with glassmorphism effects, enhanced typography, smooth animations, and improved mobile experience.

**Planned changes:**
- Add soft glassmorphism effects, gradient overlays, soft shadows (20-40px blur), increased white space, and elegant border radius (12-20px) across all pages
- Upgrade typography with bold geometric sans-serif headings (Inter/Poppins style), larger hero text (25% increase), improved line-height (1.6+), softer subtext colors, and letter spacing (0.02em-0.05em)
- Enhance hero section with background gradient blur, floating CSS shapes, smooth fade-in animation, and pill-style CTA buttons with gradient backgrounds and hover glow
- Add hover lift effect (translateY -8px to -12px), shadow depth, animated icons, gradient border glow, and 0.3s transitions to service cards
- Redesign all primary action buttons (Book Appointment, Call Now, View Doctors) to pill style with gradients, soft shadows, hover glow, and click scale animation
- Improve mobile experience with increased spacing (25%), larger touch-friendly buttons (48px minimum), sticky bottom CTA with glassmorphism, and scroll reveal animations
- Implement elegant micro-interactions including scroll reveal animations, hover transitions, button ripple/glow effects, and subtle icon animations (rotate/scale/bounce)

**User-visible outcome:** Users experience a premium, modern healthcare website with smooth animations, elegant visual depth, improved readability, and enhanced mobile interactions that feel luxurious and professional.
