# Specification

**Goal:** Add an animated "About Us" card section to the home page positioned between the header and services sections.

- Insert a new animated card section on the home page below the header and above the "Our Services" section
- Implement scroll-reveal animations using IntersectionObserver pattern consistent with existing sections

**User-visible outcome:** Visitors to the home page will see a new animated "About Us" card section with a concise clinic introduction that appears between the header and services sections with smooth scroll animations.
